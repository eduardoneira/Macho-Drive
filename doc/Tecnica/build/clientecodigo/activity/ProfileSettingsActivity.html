ProfileSettingsActivity¶

public class ProfileSettingsActivity extends AppCompatActivity¶

Actividad de modificacion de perfil.

Methods¶

changePicture¶

public void changePicture(View view)¶

Cambia la imagen de perfil del usuario (picture) No se comunica con el server, el cambio solo persiste si se pide guardar los cambios.

Parameters:
  • view –

deleteUser¶

public void deleteUser(View view)¶

Pregunta al usuario si esta seguro de que quiere eliminar su cuenta. En caso afirmativo, pide al server que la elimine y vuelve a NavigationActivity con codigo -1.

Parameters:
  • view –

onActivityResult¶

protected void onActivityResult(int requestCode, int resultCode, Intent data)¶

Metodo encargado de manejar la finalizacion de actividades. Solo se llama al seleccionar una nueva imagen de perfil. Actualiza la imagen de perfil.

Parameters:
  • requestCode – El código de la actividad que finalizó.
  • resultCode – El código resultado.
  • data – Los datos resultados.

onCreate¶

protected void onCreate(Bundle savedInstanceState)¶

Constructor de la actividad de modificacion de perfil. Inicializa username, token., name, email, picture, ubicacion Inicializa las variables mLocationListener y mLocationManager Pide al servidor la informacion del usuario y la muestra para ser modificada.

Parameters:
  • savedInstanceState –

saveChanges¶

public void saveChanges(View view)¶

Lee la informacion ingresada por el usuario y la envía al server para que sea modificada. Se actualiza la ubicacion

  • view –
