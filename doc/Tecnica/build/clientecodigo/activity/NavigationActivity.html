NavigationActivity¶

public class NavigationActivity extends AppCompatActivity¶

onOptionsItemSelected¶

public boolean onOptionsItemSelected(MenuItem item)¶

Método que responde cuando se clickea un item en el menú. Si se presionó el botón UP, se desconecta del sistema y vuelve a la actividad inicial. logOut() Si se presionó Profile Settings, se abre ProfileSettingsActivity. Si se presionó Search User, se abre UserProfileActivity Si se presionó Upload File, se abre una ventana de selección de archivo para ser subido. Si se presionó Deleted Files, se abre RecycleBinActivity.

Parameters:
  • item – El item presionado

uploadFile¶

public void uploadFile(String path)¶

Se encodea en Base 64 el archivo en el path indicado y se pide al server que lo suba. Se chequean los permisos necesarios. Se piden al usuario en caso de que falten. Permissions Se actualizan ubicacionLoc y ubicacion Llama a actualizarArchivos()

Parameters:
  • path – La ruta del archivo a subir
